# Mi Proyecto de Pizzas

Esta es una aplicación web estática y 100% local (funciona sin conexión) para gestionar recetas de pizza, inventario de ingredientes, una calculadora de masas y un diario de horneado.

## ¿Cómo usar?

1.  **No requiere instalación.** Simplemente clona o descarga este repositorio.
2.  Abre el archivo `index.html` en tu navegador web (haciendo doble clic sobre él).
3.  La aplicación guardará todos tus datos en el `localStorage` de tu navegador.

## Funcionalidades

* **Gestión de Recetas:** Guarda tus recetas de pizza favoritas, incluyendo ingredientes, pasos y fotos.
* **Inventario:** Lleva un control de los ingredientes que tienes disponibles.
* **Calculadora:** Calcula la cantidad de ingredientes necesarios (harina, agua, levadura, sal) según el número de pizzas y el porcentaje de hidratación.
* **Diario:** Mantén un registro de tus horneados, con fechas, resultados y notas.
* **Modo Oscuro:** Cambia entre modo claro y oscuro.
* **Exportar/Importar:**
    * Ve a la pestaña **Ajustes**.
    * Haz clic en "Exportar Datos (JSON)" para descargar un archivo de respaldo.
    * Usa "Importar Datos (JSON)" para cargar un archivo de respaldo previamente exportado. (¡Cuidado! Esto sobrescribirá los datos actuales).
